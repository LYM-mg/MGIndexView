# MGIndexView

### From CocoaPods【使用CocoaPods】
```ruby
pod 'MGIndexView'
```

### Manually【手动导入】
```ruby
仿写酷狗音乐的索引，只需导入分类头文件#import "MGIndexView.h"即可
```
